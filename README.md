# Apoio Solidário

Esse projeto é uma aplicação desenvolvida com `VueJs (NuxtJs)` projetada para conectar instituições sem fins lucrativos (ONGs) a potenciais doadores.

Todos os dados são requisitados pela [API do Apoio Solidário](https://github.com/apoio-solidario/apoio-solidario-back-end) desenvolvida com `Java`, `Spring Boot` e `PostgreSQL`.

## Docs

- [Documentação do Nuxt](https://nuxt.com/docs/getting-started/introduction)

## Requisitos

- Instale o [NodeJs v18](https://nodejs.org/pt/download/package-manager) ou maior;
- Instale o [VSCode](https://code.visualstudio.com/) com a extensão [Vue](https://marketplace.visualstudio.com/items?itemName=Vue.volar).

## Preparação

Certifique-se de instalar dependências:

```bash
npm install
```

## Desenvolvimento

Inicie o servidor de desenvolvimento em `http://localhost:4312`:

```bash
npm run dev
```
